# bcrypt-cli

[![Powered by skuba](https://img.shields.io/badge/ðŸ¤¿%20skuba-powered-009DC4)](https://github.com/seek-oss/skuba)

A CLI tool to generate a random key + bcrypt hash pair.

## How to Use

Install this package as an `npm global` package, and run the following:

```shell
bcrypt gen-pair
```

## Development

### Prerequisites

- Node.js LTS
- Yarn 1.x

```shell
yarn install
```

### Lint

```shell
# Fix issues
yarn format

# Check for issues
yarn lint
```

### Package

```shell
# Compile source
yarn build
```
